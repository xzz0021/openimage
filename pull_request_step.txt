## how to push request
(1)$ git pull
(2)$ git checkout -b your_branch_name
(3) make changes
(4)$ git add changes
(5)$ git commit -m "comments for changes"
(6)$ git push origin new_branch_name
(7)in github account master page, pull_request

##check current active branch on local repo 
$ git branch

## switch between your branch and the master local repo
$ git checkout your_branch_name ## switch to your branch
$ git checkout master ## switch to master local repo
